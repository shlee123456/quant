## Codebase Patterns
- Database tables use CREATE TABLE IF NOT EXISTS in _init_db() with cursor.execute() and triple-quoted SQL
- DB methods use self._get_connection() context manager (WAL mode, busy_timeout)
- Import guards follow try/except ImportError with _has_X flag pattern (see __init__.py)
- PaperTrader components: PortfolioManager, OrderExecutor, RiskManager, SignalPipeline, PerformanceCalculator
- PortfolioManager.positions and .entry_prices are direct dict attributes, .capital is direct float attribute
- CircuitBreaker constructor: CircuitBreaker(failure_threshold=5, timeout=120, expected_exception=Exception)
- SignalPipeline.process() returns Tuple[int, Optional[Any]] — unpack as (signal, regime_result)
- Broker status values: 'closed'/'filled' = filled, 'open' = pending, 'canceled'/'cancelled' = canceled
- OrderExecutor uses 5-second dedup window via _is_duplicate_order() pattern
- KIS broker: always pass overseas=True for US stocks in fetch_ohlcv() and fetch_ticker()
